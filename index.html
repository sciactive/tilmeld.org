Tilmeld

User and group manager for Nymph

Time Tested

Tilmeld is based on the Pines user manager, which ran an entire ERM. For years, it powered an e-commerce website, as well as Point of Sale systems in mall kiosks across the US. 108 Solutions ran their entire business on the code in Tilmeld.

Built for Nymph

Tilmeld is built for Nymph and provides hardened security right out of the box.

Open Source

Tilmeld is licensed under the Apache-2 license.

Flexible
